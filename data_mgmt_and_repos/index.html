    Data Management and Repositories¶

    To help HEAL investigators in the repository selection process, the HEAL Data Stewards at RENCI/RTI have developed Repository Selection Criteria and identified a number of recommended repositories. A HEAL-approved repository is a FAIR data repository that is NIH-approved and ideally has an API for metadata and data permissions calls.
